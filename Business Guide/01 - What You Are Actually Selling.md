## The Mistake Most Developers Make

Most developers say:
> "I build websites."

That is wrong. That undersells everything you do and invites clients
to compare you to a R500 Wix template.

What you actually say:
> "I build custom digital business systems that work for your business
> 24 hours a day, 7 days a week, even while you sleep."

---

## What You Delivered on Green Acorn

When you built Green Acorn Guest House, you did not build a website.
You built a complete digital business system that includes:

### Guest-Facing Features
- A Progressive Web App (PWA) installable on any phone
- Room browsing with images, pricing, and descriptions
- Photo gallery organized by category
- Events and promotions section
- Real Lekkerslaap reviews displayed professionally
- Events enquiry via WhatsApp
- Location and directions
- Offline capability — works without internet
- Nightsbridge booking integration

### Admin Panel (Owner Controls)
- Rooms management — add, edit, delete rooms and images
- Events & Promotions management — create and manage specials
- Gallery & Media management — upload and organize photos
- Menu management — update prices without touching code
- Site Settings — change hero image and text

### Technical Infrastructure
- PostgreSQL database (Supabase)
- File storage for all images
- Authentication and role-based access control
- Row Level Security on all 13 database tables
- SSL certificate (HTTPS)
- Progressive Web App with service worker
- Offline caching with IndexedDB persistence
- Image compression before upload
- Production deployment on Vercel
- Custom domain configuration (greenacorn.co.za)
- GitHub version control

### Security
- No hardcoded credentials anywhere
- Strong password policy
- File upload validation
- XSS protection
- Reverse tabnabbing prevention
- Log injection prevention

## The Three Things You Are Selling

**1. Time savings**
The owner no longer needs to answer every phone call asking about
room prices. The website answers for them.

**2. Professional image**
A professional website builds trust. Guests book with confidence.

**3. Control**
The owner controls their own content. No dependency on a developer
for every small change.
